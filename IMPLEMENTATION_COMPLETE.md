## 🎓 Техническая информация

### Как парсер определяет ИИС/ИУПСИБК

1. Читает объединённые ячейки в строке 4 листа
2. Ищет текст "ИУПСИБК" или "ИИС"
3. Сопоставляет колонки в найденными диапазонами
4. Результат: каждой группе правильно назначается её институт

### Как работает безопасная миграция

1. **Анализ** — находим всех пользователей с groupId
2. **Создание** — Institute → Direction → Program → Group (транзакция)
3. **Удаление** — DELETE Lesson для обновляемых групп
4. **Создание** — INSERT Lesson (новые данные)
5. Проверка orphaned users (пользователи со старыми группами)

console.log(`Распарсено ${records.length} записей`);

// Или сразу с безопасной миграцией
const result = await safeImportScheduleFromExcel(fileInputs, prisma);
