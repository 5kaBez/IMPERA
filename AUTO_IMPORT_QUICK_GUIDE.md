# 🚀 БЫСТРЫЙ СТАРТ: Кнопка авто-импорта в админ-панели

## ⚡ ЧТО НУЖНО СДЕЛАТЬ (30 СЕКУНД)

### Шаг 1: Откройте админ-панель
```
http://localhost:3001/admin
```

### Шаг 2: Нажмите кнопку "АВТОИМПОРТ"
```
Admin Dashboard
└─ "Импорт расписания" (раздел)
   └─ Кнопка "АВТОИМПОРТ" 
      или 
      Кнопка "АВТОИМПОРТ ИЗ GUU" 
      или "ЗАГРУЗИТЬ РАСПИСАНИЕ"
```

### Шаг 3: Ждите (1-2 минуты)
Система:
1. ✅ Скачивает файлы с guu.ru автоматически
2. ✅ Парсит их (исправленный парсер с ИИС/ИУПСИБК фиксом)
3. ✅ **СОХРАНЯЕТ всех 35 пользователей** ⚠️ ГЛАВНОЕ!
4. ✅ Загружает новое расписание
5. ✅ Создаёт новые направления (если есть)

### Шаг 4: Проверьте результат
Админ-панель покажет:
```
✅ ИМПОРТ ЗАВЕРШЁН

📊 РЕЗУЛЬТАТЫ:
- Пользователи сохранены: 35
- Групп сохранено: 42
- Новые группы: 5
- Новые направления: 2
- Уроков загружено: 9100
```

## 🎉 ВСЕ ГОТОВО!

**Все 35 пользователей видят новое расписание!**

## 📋 КАК ЭТО РАБОТАЕТ

### Раньше (старый способ):
```
❌ Нажали импорт → Потеряли пользователей → БЕД
```

### Теперь (новый способ):
```
✅ Нажали импорт → Все пользователи сохранены → ВСЕ ВИДЯТ РАСПИСАНИЕ
```

### Под капотом:

1. **Скачивание файлов** с guu.ru
   ```
   guu.ru/student/schedule/
   ├─ 1.xlsx (1 курс)
   ├─ 2.xlsx (2 курс)
   ├─ 3.xlsx (3 курс)
   ├─ 4.xlsx (4 курс)
   ├─ z.xlsx (очно-заочка)
   └─ m.xlsx (магистратура)
   ```

2. **Парсирование** (исправленный парсер!)
   ```
   ✅ Правильно разделяет ИИС от ИУПСИБК
   ✅ Поддерживает магистратуру
   ✅ Поддерживает ОЗФО
   ✅ Точно парсит уроки/преподавателей/аудитории
   ```

3. **Безопасная миграция** (главное!)
   ```
   АНАЛИЗ
   ├─ Найдено 35 пользователей
   └─ "Помечаем их как к сохранению"
   
   СОЗДАНИЕ
   ├─ Institute → Direction → Program → Group
   └─ "Новая структура создана"
   
   ОБНОВЛЕНИЕ
   ├─ DELETE старые lessons (уроки)
   └─ INSERT новые lessons
   
   РЕЗУЛЬТАТ
   ├─ ✅ 35 пользователей видят новое расписание
   ├─ ✅ Старые направления остаются для истории
   ├─ ✅ Новые направления добавлены
   └─ ✅ Бот продолжает работать!
   ```

## 🆘 ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК

### Авто-импорт не работает
1. Проверьте интернет (нужно скачивать с guu.ru)
2. Посмотрите логи: `docker logs impera-server`
3. Если сайт GUU недоступен - попробуйте позже

### Пользователи потеряли группы
**Это не должно было случиться!** Восстановление:
```
Admin → Резервные копии
├─ Найдите бекап ДО импорта
└─ Нажмите "Восстановить"
```

### ИИС/ИУПСИБК всё равно путаются
Проверьте:
1. Объединённые ячейки в строке 4 листа файла Excel
2. Убедитесь, что написано "ИУПСИБК" и "ИИС" в правильных местах
3. Если ошибка в файле GUU - свяжитесь с университетом
